# afrikaner-history
The history of the Afrikaner, from the arrival of Jan van Riebeeck through the vryburgers, the trekboere, the Groot Trek to the start of the Boere Republieke, the Vryheidsoorloë, Uniewording, First World War and the Rebellion, the Depression to the awakening of Afrikaner Nationalism; the second World War, election of 1948 to the ultimate Republic.
